# Biostatistics and Epidemiology with R

This repository contains coursework, projects, and analyses completed as part of
a Bachelor's degree in Statistics. All work is conducted in **R**, focusing on
Biostatistics, Epidemiology, and Data Analysis.

## Repository Structure

- `01_prerequisite_assignment/` – Initial Biostatistics & Epidemiology assignment
  - `data/` – Generated datasets
  - `scripts/` – R scripts
  - `report/` – Reports in R Markdown and PDF
## More to come :) 



## How to view reports

- R Markdown files (`.Rmd`) → contain the analysis code and explanations
- PDF files (`.pdf`) → ready-to-read reports
- Data files → stored in `data/` folders
